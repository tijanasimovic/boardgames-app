# Board Games Rating Web Application

This is a web application developed as a project for the **Internet Programming** course, allowing users to rate and track board games, as well as monitor their own ratings and statistics.

## Features

- User registration and login
- Browse a list of board games
- Rate games (add ratings and comments)
- View personal ratings and statistics
- Admin features (add, edit, delete games)


## Technologies

- **Frontend:** HTML, CSS 
- **Backend:** PHP/ASP.NET 
- **Database:** MySQL (managed via phpMyAdmin)
- **Tools:** VS Code, XAMPP (for local development), Postman (for API testing)
